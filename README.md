
# PMM Skills

用于沉淀和分发 PMM 场景下的 Skills。

## Skills

| Skill | 用途 |
| --- | --- |
| `logo-wall-layout` | 制作、排版、审查飞书规范的 Logo 墙。 |
| `logo-standardize` | 将 Logo 源文件标准化为白底圆角 PNG。 |

## 安装

安装 `logo-wall-layout`：

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo irisivy7421-ux/pmm-skills \
  --path logo-wall-layout

安装 logo-standardize：
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo irisivy7421-ux/pmm-skills \
  --path logo-standardize
